## North star

We are building Kittynode because we believe that:

1. The world should be shaped to serve the interests of its users.
2. Control over property should be in the hands of property holders.

There are a few work streams to achieve this goal:

1. Making it easier to be a validator by reducing the financial burden (eg. Orbit SSF).
2. Making it easier to be a validator by reducing the technical burden (eg. Kittynode).
3. Initiatives that make the validator set more representative of the users.
4. Initiatives that make the world forks (expressed through votes) in the best interest of the users.

Kittynode is focused on supporting the entire goal, which is a herculean effort. Today, we're mostly focused on making a control center for node operators.
